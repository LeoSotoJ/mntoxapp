MNTox-app

MNTox-app is a demo app for the prediction of endocrine disruptive potency in environmental samples using mass spectrometry. While annotating all MS spectra in a sample can be time consuming and impractical in most cases, MNTox applies a network-based approach to pinpoint MS/MS features with endocrine disruptive potency bypassing MS annotation.

Below, it is possible to upload your CSV file with the MS/MS features of LC-ESI-HRMS in positive mode and to predict the NR.AR endpoint potency using the MNTox network. You can also Load an example. The result is then shown in a table with highlighted rows for the alerts. The format for the CSV input file is: [ID]; [m/z]:[intensity] [m/z]:[intensity] ....

To run locally, please refer to the source code: mntoxapp
